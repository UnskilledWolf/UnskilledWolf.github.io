This is a statically generated site built using Zola.

This is my current site built using Zolla!

I found this static site generator to have a good balance between ease of use and the freedom to customize my sites. In the past, I have tried using Hugo, but I found that I was too locked into their theming ecosystem.

Although this website's design went through a few iterations, I have decided on a Sci-Fi/Cyberpunk style for it. Throughout making assets for this site, I learned that CSS actually does have a property for 9-slice textures, border-image-*! I have never seen it be used, but here, every article has a 9-slice border.
